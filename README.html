'*_wtreg.RData' are files for each case study that contain original wq data and detided DO information created in 'source_me.r' of 'detiding_cases.RProj' 'met_ls.RData' is list of metabolism estimates for each case before/after detiding, these were created using 'source_me.r' in 'detiding_cases.RProj' 'met_ls_inst.RData' is list of instantaneous flux estimates for each case before/after detiding, these were created using 'source_me.r' in 'detiding_cases.RProj', flux estimates have air-sea gas exchange already subtracted
